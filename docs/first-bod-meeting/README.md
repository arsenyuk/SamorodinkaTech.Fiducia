# Бизнес-процесс «Проведение первого заседания Совета директоров»

Первый этап жизненного цикла Совета директоров — первое (организационное) заседание, на котором избираются председатель, заместитель председателя, секретарь СД и формируются комитеты.

## Содержание

- [БЛОК 1. Подготовка (до заседания)](01-preparation.md) — задачи 1–9
- [БЛОК 2. Проведение первого заседания](02-meeting.md) — задачи 10–22
- [БЛОК 3. После заседания (результаты)](03-post-meeting.md) — задачи 23–36

Всего: **36 задач**.

## Нормативная база

- Федеральный закон от 26.12.1995 № 208‑ФЗ «Об акционерных обществах», [ст. 66–68](../laws/README.md)
- Федеральный закон от 08.02.1998 № 14‑ФЗ «Об обществах с ограниченной ответственностью»
- Федеральный закон от 06.04.2011 № 63‑ФЗ «Об электронной подписи»
- Гражданский кодекс РФ, ст. 181.2 (решения собраний)
